# COGS-300-Personal-Project
An Arduino powered arcade machine that allows you to play the snake game and dinosaur game with a joystick! The latest arduino program supports both games where you can use the other joystick to switch between game control modes and the Arduino LED matrix patterns indicate which mode you are on and displays patterns corresponding to inputs (up, down, left, right arrows + restart pattern).

# Controls
Joystick 1:
* Left = Switch to Dino game controls
* Right = Switch to Snake game controls

Joystick 2:
\Dino game
* Up = Jump
* Down = Duck
* Middle button = Restart game
Snake game
* Up = Move snake up
* Down = Move snake down
* Left = Move snake left
* Right = Move snake right
